## Installation

Install the application dependencies by running:

```sh
npm install
```

## Development

Start the server:
```sh
npm run json-server
```
and in another instance of the console:
```sh
npm run dev
```

## Unit test

Write in the console:
```sh
cd tests/
```
```sh
npm run test
```
