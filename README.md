# my-app

## Installation

Install the application dependencies by running:

```sh
npm install
```

## Development

Start the server:
```sh
npm run json-server
```
and a other console instance:
```sh
npm run dev
```

## Unit test

Execute cd /test and after that:
```sh
npm run test
```
